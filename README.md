# Project name - Child Illiteracy Analysis

# Description
A jupyter notebook analysis of child illiteracy rates across countries and the related factors. Data comes from kaggle.

# Installation:
1. Go to https://github.com/tolumorohunfola/child_literacy.
2. Click the green drop-down saying 'Code'.
3. Copy the url given from one of the options: HTTPS, SSH or GitHub CLI.
4. On your command line, navigate to the folder you would like to store the code in.
5. Paste the url


# Usage:
1. On your preferred text editor, open the terminal.
2. Ensure you have python installed which should come with pip for installing the dependencies.
3. Type the following commands:
    - pip install pandas
    - pip install numpy
4. Open the Analysis.ipynb file and run it in the terminal.
